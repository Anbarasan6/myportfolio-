Projects

Most recent work

Stock-price-movement-prediction

Stock Price Movement Prediction is a machine learning project that aims to predict whether a stock’s closing price will go up or down the next day. It uses past stock data—like the difference between high and low prices, open and close prices, and whether it’s the end of a financial quarter—to train models. Algorithms like Logistic Regression, Decision Tree, and SVM are used to classify the price direction. The final model is deployed using a web app, allowing users to input values and get real-time predictions for smarter investment decisions.

Restaurant-Recommendation-System

Restaurant Recommendation System is a user-friendly application that suggests restaurants based on two inputs: city and cuisine type. The system filters restaurant data to display a list of matching restaurants along with their ratings and cuisine categories. This helps users quickly find top-rated places in a selected location that serve their preferred cuisine. The system is built using machine learning and deployed through a clean, interactive web interface for easy access.

GitHub Repository See Live

Prime-Video-Data-Analysis-in-Power BI

This project, developed using Power BI, provides a comprehensive visual analysis of Prime Video’s content library. It explores 9,655 titles from 1920 to 2021, highlighting trends in genres, ratings, countries, and content types. The interactive dashboard offers valuable insights into viewer preferences and platform evolution over the years.
